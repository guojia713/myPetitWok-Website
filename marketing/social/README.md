# MyPetitWok — Social content kit (Instagram + Facebook)

A drafting system for the launch runway to **July 13, 2026**.
You review and post; nothing posts automatically.

## How this works

1. Each day, open `posts.md`, find today's date, copy the caption.
2. Pair it with the suggested visual (photo/video idea is in the post).
3. Post to Instagram + Facebook (same caption works for both).
4. Reply to every comment in the first hour — the algorithm rewards it.

Captions are provided in **🇫🇷 FR** and **🇬🇧 EN** (and **🇨🇳 ZH** where it fits
the Chinese-diaspora angle). Pick the language that matches the account/audience.
You're in France, so FR is usually the primary — EN broadens reach, ZH speaks to
the overseas-Chinese community who'll *love* the "小锅" angle.

## Cadence (sustainable, not spammy)

- **1 post/day** to feed is plenty. Don't burn out — 4–5/week is fine.
- **Stories daily** (lightweight: repost a recipe, a poll, a "cooking tonight").
- Save your best food video for **launch day, July 13**.
- Use **3–5 hashtags**, not 30. Mix big (#asianfood) + niche (#cuisinechinoise #recettesasiatiques).
